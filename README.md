# 💉 Carteira de Vacinação Familiar — Backend

Projeto desenvolvido para o Hackathon 1000Devs 2025 com o objetivo de gerenciar as vacinas aplicadas aos integrantes de uma família, permitindo o cadastro de pacientes, registro de imunizações, consulta de vacinas recomendadas e geração de estatísticas.

---

## 🧠 Descrição do Sistema

O sistema consiste em uma API REST responsável pelo gerenciamento do histórico vacinal dos membros de uma família. Ele permite:

- Cadastro de pacientes
- Registro das vacinas aplicadas
- Consulta do calendário vacinal por idade
- Consulta por faixa etária
- Identificação de vacinas atrasadas ou próximas
- Geração de estatísticas relacionadas às imunizações
